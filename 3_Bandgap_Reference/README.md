# 3. Bandgap Reference ‚Äî Self-Biased Current Mirror BGR

**Technology:** ASAP7 7nm FinFET | **Tool:** Ngspice + Xschem | **Analysis:** `.dc temp` sweep

---

## Objective

This module implements and characterizes a **Self-Biased Current Mirror Based Bandgap Reference (BGR)** circuit designed entirely with `asap_7nm_nfet` and `asap_7nm_pfet` FinFET devices from the ASAP7 PDK. FinFET devices operating in **deep sub-threshold** replace the BJTs that would serve as the PTAT/CTAT elements in a conventional bipolar BGR ‚Äî exploiting the exponential Ids‚ÄìVgs relationship in the sub-threshold regime to replicate diode-junction temperature behaviour at a 7nm process node.

The circuit targets a **process-independent 1.6 V reference output (vref)** with a ¬±32 mV inaccuracy budget (2%) across a ‚àí45 ¬∞C to +125 ¬∞C temperature range at VDD = 1.75 V.

## Circuit Theory & Design Equations

### Sub-threshold NMOS as a Diode-Connected BJT Surrogate

In deep sub-threshold, the NMOS drain current follows:

```
Id = I‚ÇÄ ¬∑ exp(Vgs / n¬∑Vt)
```

where `Vt = kT/q` (thermal voltage ‚âà 26 mV at 300 K) and `n` is the sub-threshold ideality factor (‚âà 1.0‚Äì1.5 for BSIMCMG at 7nm). For a diode-connected device (Vgs = Vds), this is equivalent to the V_BE of an NPN BJT. The node voltage therefore has a **negative temperature coefficient** (CTAT behaviour):

```
dVgs/dT ‚âà ‚àí1.6 mV/¬∞C   (CTAT slope from Xnfet11 ‚Üí vctat)
```

### PTAT Voltage Generation

The PTAT voltage arises from the **differential sub-threshold voltage** between two branches biased at unequal current densities. With Branch 2 using 8√ó the device area of Branch 1:

```
ŒîV_BE = Vt ¬∑ ln(N) = (kT/q) ¬∑ ln(8)
```

At T = 300 K: ŒîV_BE = 26 mV √ó 2.079 ‚âà **54 mV**

This voltage appears across R2 = 33 kŒ©, setting the branch current:

```
I_branch = ŒîV_BE / R2 = Vt ¬∑ ln(8) / R2
         = (26 mV √ó ln(8)) / 33 kŒ© ‚âà 10.7 ¬µA/branch   ‚úì
```

The temperature coefficient of the current is positive (Vt ‚àù T), making it a **true PTAT current**.

### Reference Voltage Composition

The reference branch (pfet3 + R1 + Xnfet11) combines PTAT and CTAT contributions:

```
V_ref = V_CTAT + (R1/R2) ¬∑ ŒîV_BE
      = V_CTAT + (R1/R2) ¬∑ Vt ¬∑ ln(8)
```

For first-order temperature cancellation, the PTAT slope must equal the CTAT slope magnitude:

```
(R1/R2) ¬∑ (dŒîV_BE/dT) + dV_CTAT/dT = 0
(R1/R2) ¬∑ (ln(8) ¬∑ k/q) = +1.6 mV/¬∞C
R1/R2 = 1.6 mV/¬∞C / (2.079 √ó 86 ¬µV/¬∞C) ‚âà 1.513
```

This gives R1 = 1.513 √ó 33 kŒ© ‚âà **50 kŒ©** ‚Äî exactly the value in the netlist.

At room temperature (T = 300 K):

```
V_ref ‚âà V_CTAT(300K) + (50k/33k) √ó 54 mV
       ‚âà 1.52 V + 81 mV ‚âà 1.6 V   ‚úì
```

### Current Balance Plot (Ngspice expression)

The following control-block line plots normalised branch currents to verify current mirror balance:

```spice
plot net9/30k  vref/33.33k  vctat/33.33k
```

This uses Ohm's law to back-compute branch currents from node voltages, since `v2#branch` gives only total supply current. The normalisation constants (30 kŒ©, 33.33 kŒ©) convert voltage to a current proxy for each branch.

### Temperature Coefficient of V_ref

```spice
let temp_coeff = deriv(v(vref)) / 1.24
```

`deriv(v(vref))` computes dV_ref/dT in V/¬∞C. Dividing by 1.24 (which approximates the bandgap voltage of silicon ‚âà 1.24 V at 0 K) normalises it to **parts-per-million per ¬∞C (ppm/¬∞C)** ‚Äî the standard figure of merit for reference quality. A well-designed BGR targets < 100 ppm/¬∞C; sub-threshold FinFET implementations typically achieve 50‚Äì200 ppm/¬∞C depending on mismatch.

## Startup Circuit Operation

The self-biased current mirror has a degenerate stable state at zero current (all nodes at ground or VDD with no current flowing). The startup sub-circuit ‚Äî built around **pfet5, pfet6, nfet3, nfet4, nfet5** ‚Äî detects this zero-current state and injects a small perturbation current (~1‚Äì2 ¬µA) to force the mirror into its intended operating point.

**Startup sequence:**

1. At power-up with VDD rising, `net1 ‚âà VDD` and all NMOS sources are at GND ‚Äî no current flows in the mirror.
2. `pfet5` (source = `net1` ‚âà VDD, gate = `net1`) has Vsg ‚âà 0: it is weakly on due to low Vth at 7nm, pulling `net4` toward VDD.
3. `pfet6` (source = `net2`, gate = `net1`) similarly pulls `net5`.
4. The weak injection current causes nfet3 and nfet4 (diode-connected detectors on `net5`‚Üí`net6`‚ÜíGND) to begin conducting, pulling `net1` away from VDD.
5. As `net1` drops, the PMOS mirror activates, establishing the intended 10 ¬µA/branch steady-state current.
6. Once in steady state, `pfet5`'s Vsg collapses to 0 V (source = drain = `net1`), the startup path turns off, and the circuit operates purely from self-bias.

## SPICE Analysis

### Simulation Command

```spice
.dc temp -45 150 5
```

Sweeps the global simulation temperature from **‚àí45 ¬∞C to +150 ¬∞C** in **5 ¬∞C steps** ‚Äî yielding **40 data points** across a 195 ¬∞C total span. The `.dc temp` sweep is Ngspice's built-in temperature analysis; it re-evaluates all model parameters (mobility, threshold voltage, saturation velocity, leakage) at each temperature step using the BSIMCMG thermal coefficients embedded in the model card.

### Control Block ‚Äî Plot-by-Plot Breakdown

```spice
.control
run

plot v(vref) v(vctat)
```
Overlaid temperature sweep of the reference output `vref` and the CTAT node `vctat`. The ideal result: `vref` nearly flat across the temperature axis, `vctat` showing a clear negative slope (~‚àí1.6 mV/¬∞C).

```spice
plot v(vref) - v(vctat)
```
The voltage difference V_ref ‚àí V_CTAT = PTAT contribution (V across R1). Should show a positive, approximately linear slope ‚Äî confirming PTAT character of the R1 voltage with a slope of ‚âà +1.6 mV/¬∞C.

```spice
plot v(vctat)
```
CTAT node voltage in isolation. This is the sub-threshold diode voltage of `Xnfet11`. Should decrease monotonically with temperature.

```spice
plot v(vref)
```
Reference output in isolation. Should exhibit a characteristic **bow-shaped** or near-flat curve with a minimum (zero TC point) near the centre of the temperature range. The total variation across ‚àí45 ¬∞C to +125 ¬∞C must stay within ¬±32 mV of 1.6 V.

```spice
let temp_coeff = deriv(v(vref)) / 1.24
plot temp_coeff
```
Normalised temperature coefficient in ppm/¬∞C vs temperature. The zero-crossing of this curve marks the optimal TC point. Non-zero residual at the extremes reflects higher-order curvature of the sub-threshold V_BE ‚Äî a known limitation of first-order cancellation that curvature correction techniques address.

```spice
plot net9/30k  vref/33.33k  vctat/33.33k
```
Branch current monitor via voltage/resistance proxies. All three traces should converge to approximately **10 ¬µA** at room temperature, verifying that the current mirror is balanced and the design current target is met.

```spice
plot abs(v2#branch)
```
Total supply current drawn from VDD vs temperature. Should read approximately **30‚Äì34 ¬µA** (3 branches √ó 10 ¬µA + ~2 ¬µA startup residual). Temperature dependence visible as the PTAT current increases with T.

```spice
show all
```
Prints the DC operating point summary for all devices ‚Äî useful for confirming that each PMOS is in saturation and each diode-connected NMOS is in deep sub-threshold (Vgs < Vth).
