# 3. Bandgap Reference — Self-Biased Current Mirror BGR

**Technology:** ASAP7 7nm FinFET | **Tool:** Ngspice + Xschem | **Analysis:** `.dc temp` sweep

---

## Objective

This module implements and characterizes a **Self-Biased Current Mirror Based Bandgap Reference (BGR)** circuit designed entirely with `asap_7nm_nfet` and `asap_7nm_pfet` FinFET devices from the ASAP7 PDK. FinFET devices operating in **deep sub-threshold** replace the BJTs that would serve as the PTAT/CTAT elements in a conventional bipolar BGR — exploiting the exponential Ids–Vgs relationship in the sub-threshold regime to replicate diode-junction temperature behaviour at a 7nm process node.

The circuit targets a **process-independent 1.6 V reference output (vref)** with a ±32 mV inaccuracy budget (2%) across a −45 °C to +125 °C temperature range at VDD = 1.75 V.

## Circuit Architecture

The full circuit is drawn in `bgrcircuit.sch` (Xschem 3.4.8RC) and exported to `bgrcircuit.spice`. It contains **6 PMOS** and **11 NMOS** FinFET devices plus 2 resistors, organised into five distinct functional sub-blocks.

```
VDD = 1.75 V
 │
 ├─── [Self-Biased Current Mirror — PMOS Load]
 │         pfet1, pfet2, pfet3, pfet4
 │
 ├─── [Startup Circuit]
 │         pfet5, pfet6, nfet3, nfet4, nfet5
 │
 ├─── [PTAT Branch — Branch 1]
 │         nfet1, nfet2  (SBCM NMOS legs)
 │         R2 = 33 kΩ    (PTAT voltage development)
 │         nfet6, nfet7, nfet8, nfet10  (8× diode-connected NMOS, 4 shown, 2 pairs)
 │
 ├─── [CTAT Branch]
 │         nfet11  (single diode-connected NMOS → vctat node)
 │
 └─── [Reference Branch]
           pfet3  (mirror transistor)
           R1 = 50 kΩ  (PTAT-to-VREF summing)
           nfet11 via vctat  (CTAT contribution)
           → vref output node
```

## Circuit Theory & Design Equations

### Sub-threshold NMOS as a Diode-Connected BJT Surrogate

In deep sub-threshold, the NMOS drain current follows:

```
Id = I₀ · exp(Vgs / n·Vt)
```

where `Vt = kT/q` (thermal voltage ≈ 26 mV at 300 K) and `n` is the sub-threshold ideality factor (≈ 1.0–1.5 for BSIMCMG at 7nm). For a diode-connected device (Vgs = Vds), this is equivalent to the V_BE of an NPN BJT. The node voltage therefore has a **negative temperature coefficient** (CTAT behaviour):

```
dVgs/dT ≈ −1.6 mV/°C   (CTAT slope from Xnfet11 → vctat)
```

### PTAT Voltage Generation

The PTAT voltage arises from the **differential sub-threshold voltage** between two branches biased at unequal current densities. With Branch 2 using 8× the device area of Branch 1:

```
ΔV_BE = Vt · ln(N) = (kT/q) · ln(8)
```

At T = 300 K: ΔV_BE = 26 mV × 2.079 ≈ **54 mV**

This voltage appears across R2 = 33 kΩ, setting the branch current:

```
I_branch = ΔV_BE / R2 = Vt · ln(8) / R2
         = (26 mV × ln(8)) / 33 kΩ ≈ 10.7 µA/branch   ✓
```

The temperature coefficient of the current is positive (Vt ∝ T), making it a **true PTAT current**.

### Reference Voltage Composition

The reference branch (pfet3 + R1 + Xnfet11) combines PTAT and CTAT contributions:

```
V_ref = V_CTAT + (R1/R2) · ΔV_BE
      = V_CTAT + (R1/R2) · Vt · ln(8)
```

For first-order temperature cancellation, the PTAT slope must equal the CTAT slope magnitude:

```
(R1/R2) · (dΔV_BE/dT) + dV_CTAT/dT = 0
(R1/R2) · (ln(8) · k/q) = +1.6 mV/°C
R1/R2 = 1.6 mV/°C / (2.079 × 86 µV/°C) ≈ 1.513
```

This gives R1 = 1.513 × 33 kΩ ≈ **50 kΩ** — exactly the value in the netlist.

At room temperature (T = 300 K):

```
V_ref ≈ V_CTAT(300K) + (50k/33k) × 54 mV
       ≈ 1.52 V + 81 mV ≈ 1.6 V   ✓
```

### Current Balance Plot (Ngspice expression)

The following control-block line plots normalised branch currents to verify current mirror balance:

```spice
plot net9/30k  vref/33.33k  vctat/33.33k
```

This uses Ohm's law to back-compute branch currents from node voltages, since `v2#branch` gives only total supply current. The normalisation constants (30 kΩ, 33.33 kΩ) convert voltage to a current proxy for each branch.

### Temperature Coefficient of V_ref

```spice
let temp_coeff = deriv(v(vref)) / 1.24
```

`deriv(v(vref))` computes dV_ref/dT in V/°C. Dividing by 1.24 (which approximates the bandgap voltage of silicon ≈ 1.24 V at 0 K) normalises it to **parts-per-million per °C (ppm/°C)** — the standard figure of merit for reference quality. A well-designed BGR targets < 100 ppm/°C; sub-threshold FinFET implementations typically achieve 50–200 ppm/°C depending on mismatch.

## Startup Circuit Operation

The self-biased current mirror has a degenerate stable state at zero current (all nodes at ground or VDD with no current flowing). The startup sub-circuit — built around **pfet5, pfet6, nfet3, nfet4, nfet5** — detects this zero-current state and injects a small perturbation current (~1–2 µA) to force the mirror into its intended operating point.

**Startup sequence:**

1. At power-up with VDD rising, `net1 ≈ VDD` and all NMOS sources are at GND — no current flows in the mirror.
2. `pfet5` (source = `net1` ≈ VDD, gate = `net1`) has Vsg ≈ 0: it is weakly on due to low Vth at 7nm, pulling `net4` toward VDD.
3. `pfet6` (source = `net2`, gate = `net1`) similarly pulls `net5`.
4. The weak injection current causes nfet3 and nfet4 (diode-connected detectors on `net5`→`net6`→GND) to begin conducting, pulling `net1` away from VDD.
5. As `net1` drops, the PMOS mirror activates, establishing the intended 10 µA/branch steady-state current.
6. Once in steady state, `pfet5`'s Vsg collapses to 0 V (source = drain = `net1`), the startup path turns off, and the circuit operates purely from self-bias.

## SPICE Analysis

### Simulation Command

```spice
.dc temp -45 150 5
```

Sweeps the global simulation temperature from **−45 °C to +150 °C** in **5 °C steps** — yielding **40 data points** across a 195 °C total span. The `.dc temp` sweep is Ngspice's built-in temperature analysis; it re-evaluates all model parameters (mobility, threshold voltage, saturation velocity, leakage) at each temperature step using the BSIMCMG thermal coefficients embedded in the model card.

### Control Block — Plot-by-Plot Breakdown

```spice
.control
run

plot v(vref) v(vctat)
```
Overlaid temperature sweep of the reference output `vref` and the CTAT node `vctat`. The ideal result: `vref` nearly flat across the temperature axis, `vctat` showing a clear negative slope (~−1.6 mV/°C).

```spice
plot v(vref) - v(vctat)
```
The voltage difference V_ref − V_CTAT = PTAT contribution (V across R1). Should show a positive, approximately linear slope — confirming PTAT character of the R1 voltage with a slope of ≈ +1.6 mV/°C.

```spice
plot v(vctat)
```
CTAT node voltage in isolation. This is the sub-threshold diode voltage of `Xnfet11`. Should decrease monotonically with temperature.

```spice
plot v(vref)
```
Reference output in isolation. Should exhibit a characteristic **bow-shaped** or near-flat curve with a minimum (zero TC point) near the centre of the temperature range. The total variation across −45 °C to +125 °C must stay within ±32 mV of 1.6 V.

```spice
let temp_coeff = deriv(v(vref)) / 1.24
plot temp_coeff
```
Normalised temperature coefficient in ppm/°C vs temperature. The zero-crossing of this curve marks the optimal TC point. Non-zero residual at the extremes reflects higher-order curvature of the sub-threshold V_BE — a known limitation of first-order cancellation that curvature correction techniques address.

```spice
plot net9/30k  vref/33.33k  vctat/33.33k
```
Branch current monitor via voltage/resistance proxies. All three traces should converge to approximately **10 µA** at room temperature, verifying that the current mirror is balanced and the design current target is met.

```spice
plot abs(v2#branch)
```
Total supply current drawn from VDD vs temperature. Should read approximately **30–34 µA** (3 branches × 10 µA + ~2 µA startup residual). Temperature dependence visible as the PTAT current increases with T.

```spice
show all
```
Prints the DC operating point summary for all devices — useful for confirming that each PMOS is in saturation and each diode-connected NMOS is in deep sub-threshold (Vgs < Vth).
